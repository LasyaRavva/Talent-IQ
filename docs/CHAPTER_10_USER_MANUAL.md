# Chapter 10: User Manual for Talent IQ

## 10.1 Introduction

Talent IQ is a remote interview preparation platform that combines coding practice, collaborative interview sessions, AI-based guidance, and resume analysis in one environment. This user manual provides short and practical instructions for starting the website, accessing the main modules, and using the Resume Checker to predict a resume score.

## 10.2 Getting Started

### 10.2.1 Starting the Website

To start Talent IQ locally:

1. open the project folder
2. start the backend server from the `backend` folder
3. start the frontend server from the `frontend` folder
4. open the frontend localhost address shown in the terminal

After the page loads, the user will see the Talent IQ landing page.

### 10.2.2 Accessing the Application

From the landing page, click `Get Started` or the sign-in option. After authentication, the user can access the protected areas of the platform.

### 10.2.3 Registration and Login

New users should complete the registration flow through the sign-in system. Existing users can log in directly using their credentials. After login, the user can access:

- Dashboard
- Problems
- AI Coach
- Resume Checker
- Session pages

## 10.3 Exploring the Website

The main navigation bar allows the user to move between the major sections of the platform:

- `Dashboard` for central control and session access
- `Problems` for coding practice
- `AI Coach` for concept explanation and mock interviews
- `Resume` for resume-job fit analysis
- `Services` for feature overview

For a first-time user, the best order is:

1. open the dashboard
2. practice a problem
3. use AI Coach for explanation or interview practice
4. use Resume Checker before job application

## 10.4 Using the Dashboard and Sessions

The dashboard is the main working area of Talent IQ. It allows the user to:

- create a new coding session
- join a session using a room code
- view active sessions
- review previous sessions

### Creating a Session

To create a session:

1. open the dashboard
2. click `Create Session`
3. choose a problem
4. confirm the session
5. enter the created session room

### Joining a Session

To join a session:

1. open the dashboard
2. enter the room code
3. submit the code
4. move to the related session page

## 10.5 Using the Problems Module

The Problems page is used for coding practice.

To use it:

1. click `Problems`
2. choose a problem
3. select a programming language
4. edit the starter code
5. run the program
6. check the output

This module helps users improve problem-solving skills before entering live interview sessions.

## 10.6 Using the AI Coach

The AI Coach supports two main modes:

- `Coach mode` for concept explanation
- `Interview mode` for mock interview practice

To use AI Coach:

1. open `AI Coach`
2. enter a topic
3. choose difficulty
4. select coach or interview mode
5. type or speak your prompt

In Coach mode, the AI explains concepts. In Interview mode, it asks one question at a time and gives structured feedback.

## 10.7 Predicting the Resume Score

The Resume Checker is used to compare a resume with a target role and job description. It predicts a resume-job fit score and provides detailed review feedback.

### Input Required

The user must provide:

- target role
- resume file in `DOCX` format
- job description

### Steps to Predict the Resume Score

1. open the `Resume` page
2. enter the target role
3. upload the DOCX resume
4. paste the job description
5. click `Analyze Resume`

The system will then:

- validate the file
- extract resume text
- calculate an initial score
- stream the full AI review

### Understanding the Resume Score

The score is shown as a match score out of 100. A higher score generally means better alignment between the resume and the job description. The result may also include section scores such as:

- skills match
- experience signals
- keyword coverage

The full review also includes:

- strong matches
- gaps
- risk flags
- rewrite suggestions
- verdict

The score should be treated as advisory feedback, not as a final hiring decision.

## 10.8 Viewing Previous Activity

Talent IQ supports continuity mainly through:

- past sessions on the dashboard
- recent activity panels
- downloadable resume feedback PDF

In the current version, the platform does not yet provide a full stored resume prediction-history page, but resume analysis can be saved by downloading the generated PDF.

## 10.9 Short User Guidance

For best use of Talent IQ:

1. sign in and open the dashboard
2. practice coding problems first
3. use AI Coach to clarify difficult concepts
4. use Interview mode to rehearse speaking answers
5. create or join live sessions for collaboration
6. use Resume Checker before applying for roles
